  create C-Shares : Use Case

a company can create as many share tokens as it wants. each token represents a type of share for the company

  split share : Use Case

if the value of the share rises and the company decides to split the share, the smart-contract shall allow the division of tokens by setting a single attribute and giving all shareholders the requested new amount

the split is limited to 2 decimals. it is possible to create 1.25 shares from each share but not 1.125

  determine voting rights : Use Case

The company can create multiple shares and for each set a simple voting right can be set. The only option is to allocate a certain amount of votes to each share. From 0 to n vote per share.
